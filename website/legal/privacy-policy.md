# Privacy Policy

Last Updated: May 2, 2026

## 1. Data Collection
Horizen collects minimal data required for security and moderation functionality:
- **Snowflake IDs**: We store User IDs, Guild IDs, and Channel IDs.
- **Log Data**: To provide moderation logs, we may temporarily cache message metadata (author, timestamps).
- **Configuration**: Any settings you apply via bot commands are stored in our secure sharded database.

## 2. Purpose of Collection
This data is used exclusively for:
- Operating security features (AntiNuke, AutoMod).
- Managing server-specific configurations.
- Providing moderation logging and auditing.
- Identifying whitelisted users and extra owners.

## 3. Data Retention
Data is retained for as long as Horizen is present in your server. If the bot is removed, you may request manual data deletion via our support server. AntiNuke logs and moderation history are periodically purged to ensure performance.

## 4. Data Security
We prioritize your data's safety. Our multi-database sharded backend uses industry-standard encryption and access controls to prevent unauthorized access. We never "sell" or "trade" your data to third parties.

## 5. User Rights
As a Discord user, you have the right to request a copy of the data we have stored about you or request its deletion. Please join our Support Server to initiate these requests.

## 6. Consent
By using Horizen, you consent to the collection and processing of data as described in this policy.
